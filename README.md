# Mój kantor - currency-convertor
## Jak należy obsługiwać kantor: Najpierw w pole [Kwota w zł] wpisujemy wartość (Należy pamiętać by była większa niż 1) -> Następnie wybieramy walute z pośród EURO, GBP, SEC, CHF  a potem naciskamy przycisk [Przelicz]. Pod spodem pokaże nam się kwota po operacji kantora.
